Es wurde ein Interface (IEnable) erstellt, was nicht dem eigentlichen Zweck der Aufgabe erfüllt, was mir bewusst ist.
Doch gab es Probleme mit Erstellen von Komponenten/Features, welche auf das JFrame Object zum zeichnen zugreifen muss.
Dies liegt wahrscheinlich aber nur an dem fehlenden Wissen mit JFrame.
IEnable ist somit ein Interface, welches die Funktionalität (welche in der Main Klasse implementiert wurde) des Programms wahrt.

Deswegen wurde exemplarisch ein weiteres Interface (IText) erstellt, welches je nach benutzte Komponente einen Starttext liefert.
