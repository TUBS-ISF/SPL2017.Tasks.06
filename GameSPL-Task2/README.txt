Basierend auf eine Spielerei von Früher, weswegen der größte Teil des Codes vorhanden ist.

Features die an- und abgewählt werden können:
-Obstacles, wodurch das zu überwindene Hindernis erstellt wird
-Collision, wodurch die Kollisionsabfrage de-/aktiviert werden kann

Unterschiede werden sich also bei folgenden Konfigurationen finden:
Autorunner - standart einstellung
Boring Game - ohne Hindernisse (leeres feld mit Sprungfunktion)
God Mode - Ohne Kollisionabfrage, was das "hindurchgehen" von Hindernissen erlaubt und unendlich Punkte generieren kann
